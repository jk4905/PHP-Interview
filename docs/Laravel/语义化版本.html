1.1.1. 语义化版本

版本格式：主版本号.次版本号.修订号。如 1.0.1, 3.2.39。

  1. 主版本号：当做了不兼容的 API 修改。
  2. 次版本号：当做了向下兼容的功能性更新。
  3. 当做了向下兼容的问题修复。
